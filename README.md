# Hugo Stack 博客

这是一个基于 [Hugo Theme Stack](https://github.com/CaiJimmy/hugo-theme-stack) 的博客站点，使用 [Hugo Modules](https://gohugo.io/hugo-modules/) 引入主题。

## 新建文章

仓库已提供文章模板，推荐这样创建一篇新文章：

```bash
hugo new content post/my-first-post/index.md
```

生成后直接编辑 `content/post/my-first-post/index.md`。

如果你使用自建图床，封面图和正文图片都可以直接填完整 URL。
